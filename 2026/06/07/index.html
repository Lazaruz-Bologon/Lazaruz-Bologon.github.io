  • Keep reason: Source-free + open-set 的思路对未来眼底新病灶迁移很有启发。

Source Note

同时解决 source-free 和 open-set 两个难点，用“先挑样本，再伪标注”的策略减少标注需求。

Analysis

背景：临床迁移时常见问题是：源数据不可用，目标域还可能出现源域没有的新类别。

方法：ASFOSDA 以 active learning 选择目标样本；CDU 评估 class-aware uncertainty，CPD 用 prototype discrepancy 选代表性样本；再用 target-refined self-training。

训练：在 MSD Spleen、FLARE、AMOS CT、AMOS MR 等 3D 分割任务上训练；使用 3D U-Net 和 SwinUNETR，Dice+CE 损失，source 预训练后在 target 侧自适应。

数据：3D 医学分割任务，非眼底。

创新：同时解决 source-free 和 open-set 两个难点。

实验与结果：整体优于现有方法。

局限：不是眼底任务，且主动查询仍需人工标注预算。

对眼底分割的可用性：对未来眼底新病灶、未知结构或无源数据迁移很有启发。

领域进展速览

  1. 标注效率和伪标签质量是这几天最稳定的主线。
  2. 如果后续要做新医院/新设备，XSSR 这类样本选择值得先试。
  3. 如果后续要做半监督眼底分割，Quality-Guided SSL 是最直接的加法。
