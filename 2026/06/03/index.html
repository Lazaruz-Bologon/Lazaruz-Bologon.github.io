逐篇精读

1. XSSR: Cross-Domain Self-Supervised Representative Selection for Efficient Annotation in Medical Image Segmentation

  • arXiv ID: 2606.04301v1
  • Date: 2026-06-03
  • URL: https://arxiv.org/abs/2606.04301
  • Category: 直接相关
  • Keep reason: 直接命中眼底场景：RIGA+ retinal fundus imaging，且主题是跨域标注效率。

Source Note

用源域 MAE 表征 + 目标域贪心代表性选择，在 5% 标注预算下维持分割性能。

Analysis

背景：跨设备、跨中心时，医学分割最大的成本往往不是模型容量，而是目标域标注预算。对眼底分割来说，这个问题尤其突出。

方法：先在无标注源域上训练 Masked Autoencoder 建共享 embedding；再用密度-新颖性-多样性联合评分对目标域样本做贪心选择；最后只用被选中的子集训练 U-Net。

训练：固定 5% 标注预算，并自动调节 novelty-diversity 权重。

数据：Chest X-ray、RIGA+ retinal fundus imaging、multi-site Prostate MRI。

创新：把“选哪些目标样本去标”显式变成可优化问题。

实验与结果：在 Chest X-ray 上用 22 张标注图就达到全量性能的 99.3%；Prostate MRI 上比随机选择高最多 2.5 Dice；整体比 CoreSet 高 0.4 到 1.2 Dice。

局限：仍依赖源域 embedding 的质量，fundus 只在 RIGA+ 上验证。

对眼底分割的可用性：特别适合眼底多中心新医院/新设备的优先标注排序。

2. Enhancing MedSAM with a Lightweight Box Predictor for Medical Image Segmentation

  • arXiv ID: 2606.04705v1
  • Date: 2026-06-03
  • URL: https://arxiv.org/abs/2606.04705
  • Category: 可迁移
  • Keep reason: SAM/MedSAM 的提示增强，对提示不稳定的眼底任务很有迁移价值。

Source Note

把 prompt 从“单点”升级为“点到框”的两阶段空间引导。

Analysis

背景：SAM/MedSAM 在医学图像上常被 prompt 质量限制，单点提示对小、弱对比、边界模糊目标尤其不稳。

方法：在 MedSAM 上加轻量 Box Predictor，从单点估计粗框，再把框作为额外空间先验输入。

训练：Box Predictor 先单独训练，再与 MedSAM 组合。

数据：FLARE22、BRISC、BUSI、LungSegDB。

创新：把 prompt 从单点升级为粗框。

实验与结果：相对 baseline MedSAM 在多数据集上提升稳定性和精度。

局限：当目标非常清晰时，额外框可能带来干扰。

对眼底分割的可用性：对眼底里“点提示不够准”的交互式分割很有迁移价值。

3. LightVesselNet: An Ultra-Lightweight Sub-100K Parameter Network for Retinal Blood Vessel Segmentation

  • arXiv ID: 2606.05354v1
  • Date: 2026-06-03
  • URL: https://arxiv.org/abs/2606.05354
  • Category: 直接相关
  • Keep reason: 眼底血管分割的轻量部署方向，非常直接。

Source Note

75K 参数的轻量网络，专门面向资源受限环境。

Analysis

背景：视网膜血管分割在早筛糖网和青光眼中非常关键，但很多模型太重，不适合边缘部署。

方法：紧凑 encoder-decoder，加 channel/spatial attention、多尺度 bottleneck、subpixel upsampling 和 edge residual connection。

训练：在 DRIVE、STARE、CHASEDB1、FIVES、HRF 上评估。

数据：五个公开 fundus 数据集。

创新：把部署性放到架构中心。

实验与结果：Dice 和 sensitivity 在五个数据集上都保持竞争力。

局限：跨域泛化仍需要更多验证。

对眼底分割的可用性：最适合做眼底血管分割的轻量基线。

4. Three-Dimensional Retinal Microvasculature Restoration in OCT Angiography

  • arXiv ID: 2606.05375v1
  • Date: 2026-06-03
  • URL: https://arxiv.org/abs/2606.05375
  • Category: 直接相关
  • Keep reason: OCTA 三维微血管恢复，和眼底细结构保真高度相关。

Source Note

直接恢复 3D retinal microvasculature，而不是只做降噪或投影伪影去除。

Analysis

背景：OCTA 的关键问题不是单纯噪声，而是伪影和体积内真实血管架构被 2D 处理抹掉。

方法：使用 EfficientNet-B5 encoder 和带 concurrent spatial/channel SE 的 decoder，输入相邻 3 个 B-frame，预测中间 B-frame。

训练：输入来自单个 OCTA volume 的邻近 B-frame；reference ground truth 由多次扫描平均得到。

数据：多种 vascular slabs 上开展。

创新：直接恢复 3D retinal microvasculature 的解剖结构。

实验与结果：用 PSNR、SSIM 和 Dice 评估恢复质量。

局限：依赖体积邻域输入和多次扫描平均作为参照。

对眼底分割的可用性：非常适合迁移成眼底微血管补全或 OCTA 体积重建模块。

5. Deep Learning-assisted AMD Staging based on OCT and OCT Angiography

  • arXiv ID: 2606.05379v1
  • Date: 2026-06-03
  • URL: https://arxiv.org/abs/2606.05379
  • Category: 可迁移
  • Keep reason: 病灶分割到分期的中间层设计，适合后续眼底病变流程。

Source Note

把病灶分割结果转成 biomarker maps，再用于分期。

Analysis

背景：论文目标是基于 OCT/OCTA 自动分期 AMD，重点不是直接做分割，而是把分割出的病理特征转成 biomarker maps。

方法：三条输入路线分别是 biomarker maps、2D en face OCT/OCTA、以及 3D OCT/OCTA volumes，均使用 EfficientNet-based architectures。

训练：271 participants、2030 volumes，采用 five-fold cross-validation 和标准化预处理。

数据：中心 6x6 mm OCT/OCTA volumes，共 2030 volumes 来自 271 participants。

创新：把病灶分割结果转成 biomarker maps，再用于分期。

实验与结果：比较 biomarker-based、2D 和 3D 三类模型，指标用 QWK、F1-score 和 precision。

局限：任务是分期而非分割，且对前置 segmentation 质量有依赖。

对眼底分割的可用性：适合迁移成眼底病灶分割 + 分层联合任务。

领域进展速览

  1. 标注效率和伪标签质量是这几天最稳定的主线。
  2. 如果后续要做新医院/新设备，XSSR 这类样本选择值得先试。
  3. 如果后续要做半监督眼底分割，Quality-Guided SSL 是最直接的加法。
