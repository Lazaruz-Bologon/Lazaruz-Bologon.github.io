总体判断

这批论文的主线非常清晰：一条线是围绕细结构医学分割继续做轻量化、局部细节建模和跨域泛化；另一条线是把 foundation model、active learning、source-free adaptation 这些通用方法往医学分割落地。对后续医学图像分割工作最值得优先跟踪的是轻量化部署、细边界连续性、跨域适配和伪标签质量控制这几类问题。

1. Quality-Guided Semi-Supervised Learning for Medical Image Segmentation

  • 论文 ID: 2606.01753v1
  • 归类: 直接相关
  • 保留理由: 半监督伪标签质量控制

背景

半监督分割依赖伪标签，但模型置信度并不等于真实 mask 质量，细边界和小目标尤为明显。

模型 / 算法

独立质量预测器 + quality-aware regularization + pseudo-label reweighting。

训练方法

作为可插拔模块接入已有 SSL 框架。

数据来源

作为可插拔模块接入已有 SSL 框架。

具体创新点

把“伪标签能不能信”单独建模，而不是只看 self-confidence。

实验设计

在多个医学分割设定上验证，并与常见 SSL 方法比较。

实验结果

比简单阈值筛伪标签更稳，低标注场景收益明显。

局限

依赖合成 corruption 的贴合度，真实噪声分布仍需适配。

对医学图像分割的可用性

适合低标注医学分割，是最直接的通用加法。

2. ROBUST-WT: Robust Uncertainty-aware Segmentation Transform via Whitening and Training Enhancements

  • 论文 ID: 2606.03069v1
  • 归类: 直接相关
  • 保留理由: 训练鲁棒化与不确定性建模

背景

跨设备、跨协议时，医学分割常因 scanner variation 和早期训练不稳定而掉点。

模型 / 算法

domain-adaptive augmentation、hybrid BCE+Dice、curriculum-based Dice weight scheduling 和可控 ablation。

训练方法

重点放在训练协议而不是换主干。

数据来源

重点放在训练协议而不是换主干。

具体创新点

把 scanner variation、edge noise 和 early-training instability 当作系统问题处理。

实验设计

在视盘分割基准上比较 baseline 与改进流水线。

实验结果

Dice 和 ASD 都更好，说明训练层调整确实能带来稳定收益。

局限

验证场景偏单一 benchmark，跨更多器官和模态还需补证据。

对医学图像分割的可用性

适合作为医学分割的训练配方基线。

3. XSSR: Cross-Domain Self-Supervised Representative Selection for Efficient Annotation in Medical Image Segmentation

  • 论文 ID: 2606.04301v1
  • 归类: 直接相关
  • 保留理由: 跨域样本选择与标注效率

背景

目标域标注预算往往才是医学分割的真正瓶颈。

模型 / 算法

MAE 预训练得到 shared embedding，再按 density-novelty-diversity 评分选代表样本。

训练方法

固定标注预算，自动校准样本选择权重，减少人工调参。

数据来源

固定标注预算，自动校准样本选择权重，减少人工调参。

具体创新点

把“先标哪些样本”变成可优化的问题。

实验设计

在多模态数据上与随机采样和 CoreSet 比较。

实验结果

低标注预算下依然保持稳健收益。

局限

依赖源域 embedding 质量。

对医学图像分割的可用性

适合新医院/新设备的优先标注排序。

4. Enhancing MedSAM with a Lightweight Box Predictor for Medical Image Segmentation

  • 论文 ID: 2606.04705v1
  • 归类: 可迁移
  • 保留理由: SAM/MedSAM 的提示增强

背景

SAM/MedSAM 在医学图像上常被 prompt 质量限制，单点提示对小目标尤其不稳。

模型 / 算法

在 MedSAM 上加轻量 Box Predictor，从单点估计粗框，再作为空间先验输入。

训练方法

先训练 Box Predictor，再与 MedSAM 组合，在多种医学数据集上验证。

数据来源

先训练 Box Predictor，再与 MedSAM 组合，在多种医学数据集上验证。

具体创新点

把 prompt 从单点升级为点到框的两阶段引导。

实验设计

比较 point、box 和 hybrid 配置，并做点偏移与框扩张消融。

实验结果

对 off-center click 更鲁棒，精度更稳定。

局限

单点仍然无法表达多个离散目标，目标清晰时额外框可能带来干扰。

对医学图像分割的可用性

适合提示不稳定的交互式分割场景。

5. MS-DKC: A Dataset Knowledge Card Framework for Designing and Adapting Medical Image Segmentation Models

  • 论文 ID: 2606.06103v1
  • 归类: 直接相关
  • 保留理由: 数据集知识卡与任务设计

背景

医学分割里，先选模型再看结果的做法经常忽略数据集本身的结构偏好。

模型 / 算法

把数据集特征抽成 knowledge card，再映射到模型设计与适配策略。

训练方法

对多个医学数据集做任务分析与对照。

数据来源

对多个医学数据集做任务分析与对照。

具体创新点

把数据集知识显式化，而不是留在经验里。

实验设计

对不同器官和成像场景做框架演示。

实验结果

能更清楚地说明哪些结构特征决定了模型选择。

局限

知识卡质量高度依赖人工归纳的一致性。

对医学图像分割的可用性

适合做医学分割项目立项前的数据集分析。

6. SC-MFJ: A Simple Haptic Quality Metric for Medical Image Segmentation

  • 论文 ID: 2606.06199v1
  • 归类: 可迁移
  • 保留理由: 医学分割质量度量

背景

很多分割流水线的问题不是“有没有输出”，而是“输出值不值得信”。

模型 / 算法

构建一个简单的质量度量来衡量 segmentation 结果可信度。

训练方法

作为评估/筛选模块嵌入现有流程。

数据来源

作为评估/筛选模块嵌入现有流程。

具体创新点

把质量判断从主模型里拆出来。

实验设计

与常见置信度或不确定性指标比较。

实验结果

更适合做结果筛选和伪标签过滤。

局限

仍需与具体任务的临床噪声分布匹配。

对医学图像分割的可用性

适合低标注和伪标签过滤场景。

7. MedSIGHT: Towards Grounded Visual Comprehension in Medical Large Vision-Language Models

  • 论文 ID: 2606.06760v1
  • 归类: 可迁移
  • 保留理由: 医疗大模型的 grounded comprehension

背景

医疗 VLM 如果不能把语义和空间定位接起来，就很难真正服务分割和报告。

模型 / 算法

Region Perceiver + region codebook + progressive training，把理解和定位统一起来。

训练方法

用多模态 instruction pairs 训练统一框架。

数据来源

用多模态 instruction pairs 训练统一框架。

具体创新点

把分割、定位和文本理解纳入同一建模路径。

实验设计

在多个 comprehension、grounding 和 segmentation 任务上验证。

实验结果

说明 grounded medical VLM 不只是会说概念。

局限

训练链路和 codebook 设计相对复杂。

对医学图像分割的可用性

适合和分割结果联动，做可解释的医学助手。

领域进展速览

  1. 轻量化仍然是医学分割的重要方向，尤其适合资源受限和边缘部署场景。
  2. 局部低对比、细边界断裂和结构不连续，仍然是很多细结构医学任务的主难点。
  3. 跨域样本选择、source-free 适配和伪标签质量控制，正成为医学分割里最稳的一条方法线。
  4. SAM / MedSAM、active learning、grounded VLM 的价值在于补齐提示、标注和可解释性三个薄弱环节。
