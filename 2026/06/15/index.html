1. ROBUST-WT: Robust Uncertainty-aware Segmentation Transform via Whitening and Training Enhancements

  • 原始链接: https://arxiv.org/abs/2606.03069v1
  • arXiv ID: 2606.03069v1

背景

论文明确把问题定义为多域医学分割在不同设备和临床协议下的性能退化，并把 fundus optic disc segmentation benchmark 作为主要验证场景。

模型 / 算法

在原 WT-PSE 上增加 domain-adaptive augmentation、hybrid BCE+Dice loss、curriculum-based Dice weight scheduling 和可控 ablation flags。整体思路是训练增强，而不是替换主干。

训练方法

摘要明确提到 fundus optic disc segmentation benchmark；更细的训练划分、优化器和 epoch 设置未明确说明。

数据来源

fundus optic disc segmentation benchmark。

具体创新点

把 scanner variation、edge noise、early-training instability 和 ablation control 四个问题逐项修补，形成可控训练增强框架。

实验设计

在 fundus optic disc benchmark 上对比 baseline epoch-5 与改进后 pipeline，核心结果是 Dice 和 ASD。

实验结果

最终 epoch optic-disc Dice 为 0.956、ASD 为 13.31；baseline epoch-5 Dice 为 0.939。

局限

实验主要明确落在 fundus optic disc benchmark 上，对其他眼底结构、跨设备极端域差和更复杂病灶的外推未充分展开。

对眼底分割的可用性

最适合迁移到眼底视盘/视杯分割训练协议，尤其适合替换单纯 BCE 训练并强化边界稳定性。

2. Three-Dimensional Retinal Microvasculature Restoration in OCT Angiography

  • 原始链接: https://arxiv.org/abs/2606.05375v1
  • arXiv ID: 2606.05375v1

背景

OCTA 的关键问题不是单纯噪声，而是伪影和体积内真实血管架构被 2D 处理抹掉。

模型 / 算法

使用 EfficientNet-B5 encoder 和带 concurrent spatial/channel SE 的 decoder，输入相邻 3 个 B-frame，预测中间 B-frame。

训练方法

输入来自单个 OCTA volume 的邻近 B-frame；reference ground truth 由多次扫描平均得到。摘要未明确说明优化器和损失函数。

数据来源

公开摘要未列出全部数据集名称，但比较是在多个 vascular slabs 上展开的。

具体创新点

直接恢复 3D retinal microvasculature 的解剖结构，而不是只做降噪或投影伪影去除。

实验设计

使用 PSNR、SSIM 和 Dice 评价恢复质量，并分别看 2D 与 3D microvascular fidelity。

实验结果

PSNR 从 22.23±0.78 提升到 26.16±1.26，SSIM 从 0.72±0.03 提升到 0.91±0.02；2D/3D microvascular fidelity 的 Dice 至少提升 3.8% 和 51.2%。

局限

方法依赖体积邻域输入和多次扫描平均作为参照，单帧或低信噪、跨设备偏移更强的场景未充分展开。

对眼底分割的可用性

非常适合迁移成眼底微血管补全或 OCTA 体积重建模块，用来增强细血管连续性和 3D 结构保真。

3. Local-sensitive connectivity filter (ls-cf): A post-processing unsupervised improvement of the frangi, hessian and vesselness filters for multimodal vessel segmentation

  • 原始链接: https://arxiv.org/abs/2605.21251v1
  • arXiv ID: 2605.21251v1

背景

论文把 retinal vessel analysis 视作眼病风险评估基础，目标是修复 Frangi/Hessian/vesselness 传统滤波后的血管断裂。

模型 / 算法

先计算 pixel-level vessel continuity，再用 local tolerance heuristic 填补 Frangi 响应中的 discontinuity。它本质上是无监督滤波后处理模块。

训练方法

无监督后处理，不依赖训练集；对比 naive connectivity filter、morphological closing 以及已有 literature 方法。

数据来源

OSIRIX angiographic、IOSTAR、DRIVE、STARE、CHASE-DB 等多模态/多数据集设置。

具体创新点

把血管连通性恢复前移到滤波器层，而不是事后靠复杂网络补洞。

实验设计

在多个公开数据集上比较 accuracy 和与 literature 的相对排名，同时展示无监督场景下的稳定性。

实验结果

OSIRIX 上 accuracy 最优；IOSTAR 上优于 5 篇对比工作中的 4 篇；DRIVE、STARE 上优于多篇方法；CHASE-DB 上优于所有无监督方法。

局限

作为后处理模块，它改善连通性但不改变主干网络表征；不同模态下阈值和 local tolerance 规则可能需要重调。

对眼底分割的可用性

适合直接挂到现有血管分割后处理阶段，尤其是修复断点、强化小血管连通性。

4. Uncertainty-Guided Conservative Propagation for Structured Inference in Vessel Segmentation

  • 原始链接: https://arxiv.org/abs/2605.20543v1
  • arXiv ID: 2605.20543v1

背景

论文明确指出，单次前向预测对不确定区域和断裂血管的修复能力有限。

模型 / 算法

UGCP 在 logit space 做少量 update steps，用 uncertainty-guided propagation 和 structure-aware modulation 细化输出。

训练方法

作为 plug-in module 接入不同 segmentation networks；摘要说明在 2D 和 3D vessel datasets 上验证，但未展开完整训练配方。

数据来源

四个 public vessel segmentation 数据集，覆盖 2D 和 3D 任务，摘要明确包含 retinal vessel、coronary artery 和 cerebral vessel。

具体创新点

用可靠区域支撑不确定区域，避免一锤子输出造成结构漂移。

实验设计

在四个公开 vessel segmentation 数据集上测试，并与 CNN/Transformer backbone 结合。

实验结果

Dice、centerline Dice 和 95% Hausdorff 都持续改进，同时减少 vessel disconnections。

局限

不是眼底专用方法，且迭代传播会带来额外计算；摘要没有给出具体眼底 benchmark。

对眼底分割的可用性

很适合改造成眼底血管分割的细化模块，用于不连续段修复和低置信区域传播。

审稿式判断

5. DiffSight-Former: Modeling Structural Differences and Temporal Dynamics for Glaucoma Progression Prediction

  • 原始链接: https://arxiv.org/abs/2606.09140v1
  • arXiv ID: 2606.09140v1

背景

论文明确是 glaucoma progression prediction，输入是 sequential fundus images。

模型 / 算法

结合 fundus-specific foundation model、multi-structure difference modeling 和 time-aware Transformer。

训练方法

在 SIGF 和 GRAPE 两个纵向数据集上训练；输入是随访序列 fundus images，并加入 temporal interval embeddings。

数据来源

SIGF（405 sequences）和 GRAPE（263 sequences）。

具体创新点

把 optic disc/cup 区域差异和 retinal vasculature 变化显式作为 progression signal。

实验设计

评价使用 AUC、sensitivity 和 average accuracy，并比较不同时间设置和临床 progression criteria。

实验结果

SIGF 上 AUC 为 91.54%、sensitivity 为 92.16%；GRAPE 上三种 clinical visual-field progression criteria 的平均 accuracy 为 87.48%。

局限

这不是分割任务，对单张图像的直接帮助有限；方法依赖纵向随访序列。

对眼底分割的可用性

可迁移到时序眼底分析或随访病变变化建模，作为分割后的疾病进展模块。

审稿式判断

6. Deep Learning-assisted AMD Staging based on OCT and OCT Angiography

  • 原始链接: https://arxiv.org/abs/2606.05379v1
  • arXiv ID: 2606.05379v1

背景

论文目标是基于 OCT/OCTA 自动分期 AMD，重点不是直接做分割，而是把分割出的病理特征转成 biomarker maps。

模型 / 算法

三条输入路线分别是 biomarker maps、2D en face OCT/OCTA、以及 3D OCT/OCTA volumes，均使用 EfficientNet-based architectures。

训练方法

271 participants、2030 volumes；采用 five-fold cross-validation，输入做 normalized preprocessing 并配合 data augmentation。

数据来源

中心 6x6 mm OCT/OCTA volumes，共 2030 volumes 来自 271 participants；分期标签按 AREDS simplified severity scale 划分。

具体创新点

把病灶分割结果转成 biomarker maps，再用于分层，说明结构分割可以作为病程判别的中间层。

实验设计

比较 biomarker-based、2D 和 3D 三类模型，指标用 QWK、F1-score 和 precision。

实验结果

所有模型都达到 QWK >= 0.83；biomarker-based 模型最好，QWK 为 0.85±0.03，early AMD F1 为 0.59±0.14。

局限

任务是分期而非分割，且对前置 segmentation 质量有依赖。

对眼底分割的可用性

适合迁移成眼底病灶分割 + 分层联合任务，把结构分割作为病程判别中间层。

领域进展速览

  1. 轻量化/部署方面，今天最接近落地的是 2605.21251v1，因为它是纯后处理，无需重训主干，适合塞进现有血管分割系统。
  2. 细血管或边界保真方面，2606.05375v1 和 2605.21251v1 最直接，前者修复 3D microvasculature，后者修补断裂血管连通性。
  3. 跨域/泛化方面，2606.03069v1 最有代表性，它明确把 scanner variation 和 edge noise 作为训练层问题处理。
  4. 标注效率/提示/伪标签质量方面，今天没有特别强的提示式分割新论文，但 2605.20543v1 的 uncertainty-guided propagation 可以看作对低置信区域的伪标签细化。
