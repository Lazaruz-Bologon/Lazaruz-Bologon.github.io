1. HSQ-VLM: A Novel Spatially-Constrained Quadrant Segmentation VLM Model for Explainability in Diabetic Retinopathy

  • arXiv ID: 2606.14803v1
  • Date: 2026-06-11
  • URL: https://arxiv.org/abs/2606.14803
  • Category: 可迁移
  • Keep reason: 有 fundus / quadrant segmentation / diabetic retinopathy，但核心是可解释性 VLM。

Source Note

把 fundus 划分为四象限并做具像化解读。

Analysis

背景：DR 的黑箱问题突出，论文希望把病灶和解剖位置说清楚。

方法：Landmark-Anchored Cartesian Cross-Attention + 4-quadrant Topological Latent Partitioning。

训练：3500 张高分辨率 fundus images。

数据：高分辨率 fundus 数据。

创新：不是随意分块，而是以 fovea-centered 坐标系统做结构化划分。

实验与结果：对 hemorrhages 和 microaneurysms 有高敏感度。

局限：主任务不是眼底分割本身。

对眼底分割的可用性：可作为解释性和报告生成参考。

领域进展速览

  1. 标注效率和伪标签质量是这几天最稳定的主线。
  2. 如果后续要做新医院/新设备，XSSR 这类样本选择值得先试。
  3. 如果后续要做半监督眼底分割，Quality-Guided SSL 是最直接的加法。
